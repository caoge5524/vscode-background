## English

### 📋 Workshop Categories (Planned)

| Category                 | Description                                                   |
| ------------------------ | ------------------------------------------------------------- |
| 🎬 **Background Sharing** | Share your favorite video/image backgrounds with a screenshot |
| 💡 **Tips & Config**      | Share your `settings.json` configs and theme combos           |
| 🙋 **Requests**           | Request backgrounds or ask others to share                    |
| 🐛 **Bug Reports**        | Report issues with specific media files                       |
| 💬 **General**            | Anything else about the extension                             |

### 🎥 How to Share a Background

1. Take a screenshot of VSCode with your background applied
2. Open a **new Discussion** in the `🎬 Background Sharing` category
3. You can post with this template:

```markdown
## Background Name / 背景名称

**Preview:**
[screenshot or gif here]

**Download / Source:** [link to video/image file]

**Settings used:**
{
  "vscodeBackground.theme": "glass",
  "vscodeBackground.opacity": 0.85,
  "vscodeBackground.switchInterval": 0
}

**Description:** Short description of the vibe/mood
```

### 🎥 如何分享背景

1. 应用背景后对 VSCode 截图
2. 在 **Discussions** 的 `🎬 背景分享` 分类中新建帖子
3. 可参考使用如下模板发帖：

```markdown
## 背景名称

**预览截图：**
[在此放置截图或 GIF]

**下载/来源：** [视频/图片文件链接]

**使用的配置：**
{
  "vscodeBackground.theme": "glass",
  "vscodeBackground.opacity": 0.85,
  "vscodeBackground.switchInterval": 0
}

**描述：** 简短描述这个背景的氛围/风格
```
